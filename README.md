# 🐱 DenLib — Менеджер пакетов для Python и Node.js

**DenLib** — это автономный установщик библиотек из PyPI и npm. Работает без установки Python, pip, npm или любых других зависимостей. Всё упаковано в один файл.

## 📦 Что в репозитории?
Только необходимое для пользователей:
- `setup.exe` — установщик 
- `README.md` — эта инструкция
- `LICENSE` — лицензия MIT


## 📥 Установка
1. Скачай **`setup.exe`** из репозитория или релизов.
2. Запусти файл.
3. Нажми **«📥 Установить DenLib»**.
4. Программа сама распакует основной движок, настроит путь в системе и выведет сообщение об успехе.

>  Перезапусти терминал (cmd / PowerShell), чтобы команда `denlib` стала доступна.

## 🚀 Использование (пример)
Открой терминал и введи:
```bash
denlib install npm express   # Установить библиотеку из Node.js
denlib install pip requests  # Установить библиотеку из PyPI
denlib list                  # Показать уже установленные пакеты
